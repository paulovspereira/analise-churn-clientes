# 📊 Projeto: Análise de Churn de Clientes

## 👨‍💻 Autor

Paulo Pereira – Transição de carreira para Analista de Inteligência Artificial Júnior Júnior

---

## 📌 Objetivo

Identificar clientes com maior risco de cancelamento para apoiar estratégias de retenção.
---

## 📂 Dataset

* Dados simulados de clientes bancários
* Variáveis analisadas:

  * Tempo de conta
  * Fidelidade
  * Tipo de contrato
  * Tipo de serviço

---

## 🛠️ Tecnologias

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Statsmodels / Scikit-Learn
* Jupyter Notebook

---

## 🔎 Etapas do Projeto

### 1️⃣ Limpeza de Dados

* Tratamento de valores nulos
* Conversão de tipos
* Criação de variáveis categóricas

### 2️⃣ Análise Exploratória (EDA)

* Distribuição de clientes
* Churn por tipo de contrato
* Correlação entre variáveis

### 3️⃣ Modelagem

Regressão logística para identificar fatores de churn.

Principais resultados:

* Contratos de 5 e 20 anos reduzem fortemente o churn.
* Contas PJ e poupança têm menor risco.
* Tempo de conta tem impacto pequeno.

---

## 📊 Insights de Negócio

✔ Incentivar contratos longos
✔ Criar programas de fidelidade
✔ Oferecer benefícios para contas novas
